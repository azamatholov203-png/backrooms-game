# Projects Repository

## 1. 🎮 THE BACKROOMS: REALITY COLLAPSE

HTML Canvas асимметричный кооперативный психологический хоррор с системой искусственного интеллекта.

**Файл:** `index.html` + `game.js`

### 🎯 Функции:
- ✅ Психологический распад разума (Некроз)
- ✅ 4 уникальных класса персонажей (Реактивный, Глитч, Ученый, Нормальный)
- ✅ ИИ-монстр Бактерия с механикой мимика
- ✅ Система миндальной воды (10% шанс заражения)
- ✅ Ловушки на карте
- ✅ HUD с показателями психики, здоровья и расстояния до союзников
- ✅ Инвентарная система

### ⚙️ Механики:
- **Некроз Разума**: Изоляция убивает вас. 10+ минут = инвертированное управление. 15+ минут = полный распад.
- **Социализация**: Стояние рядом с товарищем восстанавливает психику.
- **Инфекция**: Зараженные игроки становятся врагами (красные).
- **Мимик**: Монстр копирует голоса убитых игроков.

### 🎮 Управление:
- **WASD** - Движение
- **SPACE** - Взять воду
- **Q** - Пить воду
- **E** - Прыжок
- **LMB/RMB** - Атака

## 2. 🌤️ WEATHER DASHBOARD

Интерактивный профессиональный метеорологический дашборд с интеграцией Open-Meteo API.

**Файл:** `weather-dashboard.html`

### 🌍 Особенности:
- ✅ **Поиск по городам** - вводите название любого города в мире
- ✅ **Текущая погода** - температура, влажность, давление, UV-индекс, ветер
- ✅ **Почасовой прогноз** - 24 часа с погодными иконками
- ✅ **7-дневный прогноз** - макс/мин температура, осадки, скорость ветра
- ✅ **Детальная информация**:
  - Скорость ветра (км/ч)
  - Осадки (мм)
  - Атмосферное давление (hPa)
  - UV индекс с категориями (Low/Moderate/High/Very High/Extreme)
- ✅ **Избранные города** - 6 популярных городов для быстрого доступа:
  - London 🇬🇧
  - New York 🇺🇸
  - Tokyo 🇯🇵
  - Paris 🇫🇷
  - Sydney 🇦🇺
  - Dubai 🇦🇪
- ✅ **Красивый дизайн** - градиенты, адаптивный макет, плавные переходы
- ✅ **API без ключей** - полностью бесплатно!
- ✅ **Автообновление времени** - показывает когда были загружены данные

### 🛰️ API Источники:
1. **Open-Meteo Geocoding** - перевод названия города в координаты
2. **Open-Meteo Forecast** - получение погодных данных
   - Оба полностью бесплатны, без лимитов и без регистрации

### 🎨 Интерфейс:
- Purple gradient фон
- Белые карточки с теневыми эффектами
- Responsive дизайн (работает на мобильных)
- Плавные переходы и анимации
- WMO погодные коды с эмодзи иконками

### 📊 Показатели:
- Current Conditions (температура, влажность, давление, ветер, UV)
- Hourly Forecast (24 часа вперед)
- Daily Forecast (7 дней)
- Precipitation & Wind Statistics
