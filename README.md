# THE BACKROOMS: REALITY COLLAPSE

## Психологический HTML Canvas Хоррор

Интерактивная игра выживания в бесконечных жёлтых лабиринтах Уровня 0 Backrooms.

### 🎮 ОСОБЕННОСТИ

- **Психологический распад разума (Некроз)** - изоляция убивает ваш разум
- **Система инфекции** - миндальная вода может быть заражена
- **Уникальные классы** - Реактивный человек, Глитч-человек, Ученый, Лунатик, Клептоман
- **Асимметричный геймплей** - контролируйте выжившего ИЛИ ИИ-монстра
- **VHS эстетика** - классический аналоговый стиль 1980-х
- **Кооперативная механика** - социализация спасает вас от безумия

### 🕹️ УПРАВЛЕНИЕ

- **WASD** - Движение
- **SPACE** - Взаимодействие с водой
- **Q** - Пить воду из инвентаря
- **E** - Прыжок
- **LMB/RMB** - Атака

### 📊 МЕХАНИКИ

#### Некроз Разума
- Одиночество убивает вас
- 10+ минут изоляции - управление инвертируется
- 15+ минут - полный распад психики
- Спасение - стойте рядом с живым товарищем

#### Миндальная Вода
- Единственный ресурс для лечения безумия
- 10% шанс заражения при питье
- Можно получить из ванн или трупов

#### Монстр (Бактерия)
- Спавнится через 10 минут игры
- Копирует голоса убитых игроков
- Использует механику мимика с тестом Тьюринга

#### Физиологические дебаффы
- **Впечатлительный** - не может разделывать трупы
- **Аллергик на миндаль** - вода смертельна, только социализация
- **Реактивный** - радиация, но взрыв уничтожает вирус
- **Глитч-человек** - не деградирует в одиночестве, видит сквозь стены

### 🎯 ЦЕЛЬ

- **Выживание** - продержаться максимально долго
- **Побег** - договориться с мародерами и активировать портал
- **Очищение** - уничтожить Бактерию и спасти команду
